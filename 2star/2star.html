Improving our Shooting Star

For the 1 Star tasks we created a shooting star particle system. Now we will look to improve it to create a more detailed effect

Task 1: Editing Values

To create more detail we want to have multiple particle systems. To make this process more effecent we should use the same class but look to explore some of the values

Edit the particle class so that the following values are applied through the constructer when a new instnance is insilised:

      1. Colour

      2. lifespan

      3. Make sure to update the class to make use of the new colour property

Task 2: Layer Particle Systems

Next we want to add multiple systems so that we can create a layered effect with a few different colours and lifespans. This will help to create a more detailed look

To achieve this do the following:

      1. In sketch.js create an instance called orangeParticles set the value of colour to be a orrange of your choice

      2. In sketch.js create an instance called redParticles set the value of colour to be a red of your choice

      3. edit sketch.js to apply the two new systems in a similar way to the existing system

You will now have a amore detailed particle system that looks closer to a stream of smoke and fire
