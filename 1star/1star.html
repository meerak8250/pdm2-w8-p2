Creating a Shooting Star

For the Star 1 tasks we are going to edit the particle system to create a shooting star

The 1 Star tasks have two .js files sketch.js which uses our system and particle.js which creates each particle

As we saw in the lecture, this system works by spawning new particles and then removing them after a set amount of time.

For the 1 Star tasks, we will look to edit the values of our particle to change how our system looks

Task 1: Starting Position

When looking at our system, we can see that it appears to spill out and fall downwards. There are two parts of the particle class that cause this to happen.

The first part is the velocity on line 5 which sets out initial starting position

The second part is gravity on line 10 which sets the direction out particle falls in

For this task, edit line 5 and 10 so that the following happens:

      1. Change the values on line 5 so that the particles are flowing towards the right bottom corner of the sketch

      2. Make sure that the particles remain closely packed together (they do not spread out too far)

Next we need to edit the gravity value:

      On line 10, edit the Vector values to increase the speed of the particles by increasing gravity. The speed is up to you

      Make sure the particles still flow towards the bottom right corner of the canvas

Task 2: lifespan

For this particle system we want the particles to disappear before they go off the screen

To achieve this do the following:

      1. Edit the lifespan value on line 6 to make the particles fade out before they leave the screen

      2. Change line 29 to be an if statement. When the lifespan is less than half of its starting value begin to fade out the particle, else draw the particle with no fade

Task 3: Colour

The final change required is to alter the colour of our particles

We can do this by:

      1. Edit the fill() value to be a yellow colour. fill(240, 240, 60); works well

    You will now have a particle system that sprays yellow sparks towards the bottom right of the screen, they should disappear halfway between the starting position and the bottom right corner
